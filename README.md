# Maxxs Casino

A stylish, single-page blackjack game with chip betting, animations, and celebratory effects.

## Play Locally

Open `index.html` directly in your browser, or run a local server:

```bash
python3 -m http.server 5173
```

Then visit:

```
http://localhost:5173/
```

## Features

- Chip betting with drag or click-to-bet animations
- Side bets + settings
- Dealer/player animations and effects
- Confetti + balloon celebration on wins/blackjack
- Developer mode for testing outcomes (password protected)
